## Sponsor Tools Used

### ElevenLabs

ElevenLabs powers our AI voice agent for patient intake calls. We use the ElevenLabs Conversational AI platform with a custom voice agent that conducts structured medical intake interviews over the phone. The agent uses dynamic prompts to guide patients through questionnaires, collects their responses via speech recognition, and the results are stored in FHIR-compliant format. The voice agent is configured with a professional, empathetic tone suitable for healthcare interactions.

### Additional Tools

- **Twilio**: Phone number provisioning and call routing to connect patients to the ElevenLabs voice agent
- **Aidbox**: FHIR R4 server for healthcare data storage and interoperability
